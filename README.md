# 🧠 AI That Learns You

> A personalized study intelligence system — predicts productivity using ML, analyses your PDFs, generates MCQ quizzes, and evaluates your study technique using AI.

---

## 📁 File Structure

```
AI-Learns-You/
│
├── data/
│   └── study_data.csv          ← Training dataset (hours, focus, sleep, subject, productivity)
│
├── src/
│   ├── train.py                ← Train + save the Random Forest model
│   ├── pdf_analyzer.py         ← PDF text extraction utility (used by app.py)
│   ├── quiz_generator.py       ← MCQ generation logic (used by app.py)
│   └── technique_checker.py    ← Study technique ratings logic (used by app.py)
│
├── models/
│   ├── model.pkl               ← Trained Random Forest model (auto-generated by train.py)
│   ├── feature_importance.csv  ← Feature importance scores (auto-generated)
│   └── metrics.json            ← R², MAE, CV scores (auto-generated)
│
├── app.py                      ← Main Streamlit dashboard (run this)
├── requirements.txt            ← Python dependencies
└── README.md                   ← This file
```

---

## ⚡ Quick Start

### 1. Clone the repo
```bash
git clone https://github.com/lord-of-masquerade/AI-Learns-You.git
cd AI-Learns-You
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Set your Anthropic API key (for AI features)
```bash
# macOS / Linux
export ANTHROPIC_API_KEY=sk-ant-...

# Windows CMD
set ANTHROPIC_API_KEY=sk-ant-...
```
Get your key at https://console.anthropic.com

### 4. Train the model
```bash
python src/train.py
```
This creates `models/model.pkl`, `models/feature_importance.csv`, and `models/metrics.json`.

### 5. Run the app
```bash
streamlit run app.py
```

## 🛠 Features

| Feature | Description |
|---|---|
| ⚡ **Productivity Predictor** | Random Forest predicts your study score from hours, focus, sleep, distractions, subject |
| 📊 **Session History** | Logs all predictions with line chart over time |
| 📄 **PDF Study Analyser** | Upload notes/textbook → Claude extracts key concepts, weak areas, study order |
| 🧩 **MCQ Quiz Generator** | Select topic or paste PDF → Claude generates fresh MCQs with explanations |
| 🔬 **Study Technique Checker** | Select technique(s) + describe routine → science-backed ratings + Claude feedback |

---

## 📊 Dataset columns

| Column | Type | Description |
|---|---|---|
| `hours_studied` | int | Hours in the study session |
| `focus_level` | int | Self-rated focus (1–10) |
| `distractions` | int | Distraction level (0–10) |
| `sleep_hours` | int | Previous night's sleep |
| `subject` | str | DSA / OOP / Maths / Physics / History |
| `productivity` | int | Target: productivity score (1–10) |
